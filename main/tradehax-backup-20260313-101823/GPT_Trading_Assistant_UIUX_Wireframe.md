# GPT Trading Assistant UI/UX Wireframe

## Dashboard
- Signal/alert feed: Real-time, sortable, filterable
- Action buttons: Execute trade, set alert, view audit
- Portfolio summary: Performance, risk, positions
- Compliance status: Audit trail, regulatory notes

## Signal Details
- Confidence score, reasoning, factors (momentum, sentiment, volatility, macro/micro)
- Historical win rate, backtest results
- Alert configuration: Thresholds, notification channels

## Trade Execution
- Simple form: Symbol, side, quantity, price, signalId
- Confirmation modal: Audit log preview, compliance check

## Portfolio View
- Performance metrics: Sharpe, VAR, win rate, drawdown
- Position list: Symbol, quantity, avg price, PnL
- Trade history: Executed trades, signals, audit events

## Compliance View
- Audit event list: Filter by type, severity, actor
- Proof chain visualization
- Downloadable reports
